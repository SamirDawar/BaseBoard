### KiCad History

KiCad was originally developed by Jean-Pierre Charras. In 2013, CERN's BE-CO-HT section (Beams Department, Controls group, Hardware Technology section) started contributing to KiCad and provided resources to support its development. KiCad is now maintained by a global community of developers.

## Our Modifications

This fork ("KiCad AI Fork" / "Spring Break CAD Project") adds AI-powered features for intelligent circuit design and PCB layout optimization. All modifications are:

- Licensed under GPL-3.0 (same as upstream)
- Documented in our git history
- Clearly marked with version suffix "-AI-Fork"
- Designed to be potentially contributed upstream where appropriate

### Key Modifications

1. **IPC API Integration** - External AI service communication layer
2. **ML-Enhanced Autorouter** - Machine learning-based trace routing
3. **Smart Component Placement** - AI-optimized component positioning
4. **Design Assistant** - AI-powered design recommendations
5. **Training Data Pipeline** - System for learning from existing designs
